## Installation/Configuration
Cloner le projet : ```git clone https://github.com/constantinlfg/game-library```

Installer les dépendances : ```composer install```

## Présentation du projet
Il s'agit d'une webapp créer en binome lors d'un projet universitaire. Nous devions créer une webapp de présentation de jeux vidéo. La base de données nous a été donnée. L'application a été codé en PhP pure sans framework. Le projet s'est étalé sur 1 semaine de travail.
